# Exercício: Two-Way Data Binding em React Native

**Objetivo:**  
Pesquise o que é **Two-Way Data Binding** e crie um pequeno exemplo em **React Native** que demonstre esse conceito.  

**Tarefas:**  
1. Pesquise e explique com suas palavras o que significa **Two-Way Data Binding**.  
2. Crie um aplicativo simples em React Native com:  
   - Um campo de texto (`TextInput`) que atualize o estado do aplicativo enquanto você digita.  
   - Um componente (`Text` ou lista) que mostre o valor do estado em tempo real.  
3. Envie seu código e uma breve explicação de como o Two-Way Data Binding está funcionando no seu exemplo.  

**Dica:**  
Use o hook `useState` para gerenciar o estado do input.
